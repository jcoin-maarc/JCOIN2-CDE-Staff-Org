# JCOIN 2.0 Common Data Elements: Staff / Organizational Measures

This repository contains the most up-to-date JCOIN 2.0 Staff / Organizational CDE resources. 

Each update to the CDE resources is tracked with a version and date stamp along with a document that provides an outline of changes introduced in each version and instructions on implementing these changes at the hub level.

## Resources: Baseline Staff / Organizational CDEs
The repository includes the following resources for Baseline Staff / Organizational CDEs:

- **Formatted user-friendly reference document**: jcoin2_staff_org_cde_items_baseline_<version_date>.pdf
- **REDCap project user protocol / guidelines**: jcoin2_staff_org_cde_redcap_protocol_<version_date>.pdf
- **REDCap codebook**: jcoin2_staff_org_cde_redcap_codebook_<version_date>.pdf
- **REDCap data dictionary**: MAARCJCOIN20OrganizationalStaf_DataDictionary_<version_date>.csv
- **Explanation to the REDCap data dictionary columns**: jcoin2_redap_data_dictionary_column_descriptions.pdf
- **REDCap Staff/Org CDE project version change log along with implementation instructions**: jcoin2_staff_org_cde_track_changes_<version_date>.pdf
